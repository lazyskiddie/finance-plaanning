# Key Features
Income Input: A field to enter the user's total monthly income.

Two Budgeting Scenarios: Users can choose between two scenarios:
   - With an Ongoing Loan (EMI): Allocates a portion of income specifically for a monthly loan installment.
   - Without an Ongoing Loan (EMI): Distributes the income across other categories.
